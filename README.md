# Dead Code Detector

Identifies and flags potentially dead code in the codebase.

## Free
```yaml
- uses: walshd1/dead-code-detection@v1
  with:
    gemini_api_key: ${{ secrets.GEMINI_API_KEY }}
```

## Paid (cost + 4.75%)
```yaml
- uses: walshd1/dead-code-detection@v1
  with:
    service_token: ${{ secrets.ACTION_FACTORY_TOKEN }}
```
